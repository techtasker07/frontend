# MIPRIPITY - Property Management & AI Prospect Analysis Platform

## Project Overview
MIPRIPITY is a modern web application built with Next.js 14 and Supabase that serves as a comprehensive property management platform with AI-powered investment analysis features. The platform enables users to list properties, participate in community voting, and leverage AI technology for property investment prospect analysis.

## Core Concepts & Purpose
- **Property Management**: A community-driven platform where users can list, browse, and evaluate real estate properties
- **Community Voting**: Users can vote on properties with category-specific voting options to provide feedback and insights
- **AI Prospect Analysis**: Advanced AI features that analyze property images and generate investment prospects with detailed cost breakdowns and implementation roadmaps
- **Investment Decision Support**: Tools to help users make informed property investment decisions through data-driven insights

## Technology Stack

### Frontend
- **Framework**: Next.js 14.2.25 (React 18.2.0)
- **Styling**: Tailwind CSS with custom components
- **UI Components**: Radix UI primitives with custom design system
- **Animations**: Framer Motion for smooth transitions
- **Forms**: React Hook Form with Zod validation
- **Charts**: Recharts for data visualization
- **Icons**: Lucide React
- **Notifications**: Sonner for toast notifications

### Backend & Database
- **Backend-as-a-Service**: Supabase
- **Database**: PostgreSQL with Row Level Security (RLS)
- **Authentication**: Supabase Auth
- **File Storage**: Supabase Storage for property images
- **Real-time**: Supabase real-time subscriptions

### AI & ML
- **Image Analysis**: Teachable Machine and TensorFlow.js
- **AI Processing**: Custom algorithms for property categorization and prospect generation
- **Smart Analysis**: Intelligent property type detection and investment prospect creation

## Key Features & Functionalities

### 1. User Authentication & Profiles
- **Registration/Login**: Email-based authentication with Supabase
- **Profile Management**: User profiles with first name, last name, email, phone, and profile picture
- **Protected Routes**: Route protection for authenticated users
- **Session Management**: Persistent login sessions with automatic redirects

### 2. Property Management System
- **Property Listing**: Users can create detailed property listings with:
  - Title, description, location
  - Property category (Residential, Commercial, Land, Industrial, Materials, Mixed-Use)
  - Current worth estimation
  - Year of construction
  - Contact phone number
  - Multiple image uploads

- **Property Browsing**: Advanced property discovery with:
  - Grid and list view modes
  - Category filtering
  - Search functionality (title, location, description)
  - Sorting options (newest, oldest, price high/low, alphabetical)
  - Pagination for large datasets

- **Property Details**: Comprehensive property view with:
  - Image gallery
  - Owner information
  - Vote statistics and breakdowns
  - Edit/delete functionality for owners

### 3. Community Voting System
- **Category-Specific Voting**: Different voting options for each property category:
  - **Residential**: Rent Out, Sell, Renovate, Too Expensive, Good Location, etc.
  - **Commercial**: Lease, Sell, High Traffic, Poor Access, Profitable, etc.
  - **Land**: Build House, Farming, Commercial Use, Hold & Sell, etc.
  - **Industrial**: Factory Use, Warehouse, Convert, High Demand, etc.
  - **Materials**: Bulk Buy, Quality, Overpriced, Scarce, etc.
  - **Mixed-Use**: Lease, Sell, Convert, High Demand, etc.

- **Vote Analytics**: Detailed statistics showing:
  - Vote distribution percentages
  - Total votes per property
  - Community sentiment analysis

### 4. AI Prospect Analysis (Core Feature)
- **Post-Login AI Trigger**: Automatic AI feature activation after successful login
- **Image Capture**: Camera access and file upload for property images
- **Smart Categorization**: AI-powered property type detection
- **Prospect Generation**: Automatic creation of 5 investment prospects per analysis
- **Detailed Analysis**: Each prospect includes:
  - Investment description
  - Development cost breakdown
  - Total investment required
  - ROI potential assessment
  - Implementation roadmap with step-by-step tips

### 5. Dashboard & Analytics
- **Personal Dashboard**: User-specific overview with:
  - Property portfolio summary
  - Voting activity statistics
  - Performance analytics

- **Platform Analytics**: Community-wide statistics:
  - Total users, properties, votes
  - Recent activity feed
  - Category distribution
  - Vote performance metrics

### 6. Responsive Design & UX
- **Mobile-First**: Optimized for mobile devices with touch interactions
- **Responsive Layout**: Adaptive design for all screen sizes
- **Sidebar Navigation**: Hover-activated sidebar on desktop, slide-out on mobile
- **Progressive Web App**: PWA-ready with proper manifest and icons

## Database Schema

### Core Tables
- **profiles**: User profile information (extends auth.users)
- **categories**: Property categories (Residential, Commercial, etc.)
- **vote_options**: Category-specific voting options
- **properties**: Main property listings with full details
- **property_images**: Image storage for properties
- **votes**: User votes on properties
- **prospect_properties**: AI-analyzed properties for investment prospects
- **property_prospects**: Individual investment prospects with cost analysis

### Key Relationships
- Properties belong to users and categories
- Votes link users to properties with specific vote options
- Property prospects are generated for prospect properties
- Images are associated with properties

## Security & Permissions

### Row Level Security (RLS)
- **Public Read**: Properties, votes, categories accessible to all
- **Owner Write**: Users can only modify their own properties
- **Authenticated Create**: Only logged-in users can create content
- **Vote Prevention**: One vote per user per property

### Authentication Guards
- **Protected Routes**: Automatic redirects for unauthenticated access
- **Session Validation**: Continuous authentication state checking
- **Secure API**: All API calls require valid authentication
